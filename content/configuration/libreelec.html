LibreELEC

It is easy to confuse LibreELEC with Kodi itself and several areas on this site will overlap. LibreELEC is the operating system - the software that starts when the computer is turned on. Kodi is the media application that is pre-installed on LibreELEC.

Installation

Installing LibreELEC is done by downloading the current version from the LibreELEC website and writing the image to a high-quality thumbdrive. The easiest way to write the image is to use the LibreELEC USB-SD Creator from the LibreELEC downloads page. This tool will automatically download the selected version and write it to the thumbdrive you choose. Alternatively, you can download the image and write it to a USB drive manually in Linux.

After flashing the image the thumbdrive is plugged into the Beelink Mini S12 Pro mini desktop computer. The computer boots off the flash drive when powered-on and guides you through installation.

Configuration

Getting LibreELEC installed and Kodi running takes answering just a few questions. During installation you'll be prompted to set a Hostname. The naming convention for computer host names used on my network are "XBMC" and a date code indicating the date of installation.

The following table lists the host names for the Kodi boxes installed.

XBMC-2308 Beelink mini computer in living room installed in August 2023.
XBMC-2309 Beelink mini computer in bedroom installed in September 2023.
XBMC-2405 Beelink mini computer in Charlotte installed in May 2024.

Until a few years ago Kodi was called XBMC which stood for XBox Media Center. The named was changed a few years ago for copyright reasons.
