Kodi

After installing LibreELEC and completing the initial setup of Kodi you can tailor Kodi to your needs and add content. The following explains how to configure Kodi to play media on TVs.

Appearance

Two things make Kodi the best media player for TVs: first, it can play literally any type of media file; second, you can make it look almost anyway you wish using "skins". The default skin for Kodi is a blue-black colored skin named "Estuary". This skin works with any add-on installed from the Kodi official repository.

The skin of choice for all Kodi boxes is the Amber skin. This skin provides a nice balance between simplicity and features found in more advanced (and resource-intensive) skins. The skin has extensive settings that can be modified to change the behavior and appearance. After installing the skin copy the settings.xml file from another working Kodi box or from a recent backup of XBMC-2308. The location of the settings.xml file in Kodi is: /storage/.kodi/userdata/addon_data/skin.amber/settings.xml

If something in Kodi stops working (other than playing media) or an add-on's configuration page is missing change the skin back to default and attempt to make the change. Once resolved you can change the skin back.

Add-Ons

Another great feature of Kodi is it's support for add-ons that extend the functionality or change the behavior of Kodi. Some add-ons are created by the Kodi Team (the developers and maintainers of the Kodi software) while many more are created by regular Kodi users.

The following table lists the add-ons installed on the Kodi boxes.

Backup (script.xbmcbackup) Provides scheduled backups and restoring of the Kodi databases, add-ons and add-on data, playlists, etc.
Trakt (script.trakt) Scrobbler for TV shows and movies to the website Trakt.tv. Although not needed the information provided by Trakt is interesting and has made it easier to mark specific episodes as watched in Kodi after new installs.
Youtube (plugin.video.youtube) Allows Kodi to play Youtube videos (without ads) directly from Kodi or through the use of the browser plugin Play to Kodi from other devices.
Note: the Google account used is one specifically created for the TVs.
Last.fm (service.scrobbler.lastfm) Service that scrobbles music played to Last.fm. The plugin scrobbles tracks to the account "jchaven" with some directories excluded.
Repositories

Add-ons are installed from online software sources called Repositories or from software packages manually downloaded from the developer's website using the Install from Zip option in Kodi. As a general rule add-ons should only be installed from the official Kodi repository or from well-trusted developers.

Illegal Add-Ons

Kodi is often described by news organizations and law enforcement as a player for pirated media or aligned with online piracy outlets. Although Kodi can play pirated media (like legally-sourced media) it is not a tool to download or play pirated media. However, several plugins have been created that can find and play pirated media or pirated media streams. The Kodi Team calls these illegal addons and will not offer support in forums if any illegal addons are listed in uploaded logs. Almost all requests for support require a full debug log be uploaded to the forum to receive help. Additionally, Kodi just works better when the number of add-ons is limited and only installed from official repositories.

Sources

The content that Kodi plays and the type of content (TV, Movies, etc.) is defined by adding the directories where the media is located the sources for TVs, Movies, or Music. Editing sources Kodi can be done using the settings menu or by editing the XML file directly.

Editing XML File

Editing sources using the GUI is quite easy and fool-proof. However, if your source has many directories it can be arduous and time consuming to add all the directories to the source. An easier way is to edit the XML file directly. The XML file that Kodi uses to define sources is: /storage/.kodi/userdata/sources.xml

The following is the contents of the sources.xml file on one of the Kodi boxes.
